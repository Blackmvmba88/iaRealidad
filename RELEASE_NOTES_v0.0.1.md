## ğŸŒŸ What is iaRealidad?

**iaRealidad** is a revolutionary cross-platform AR (Augmented Reality) assistant for electronics repair, measurement, and creation. It transforms your mobile device into an intelligent repair companion that:

- **Sees** circuit boards and identifies components in real-time
- **Listens** to electrical noise and mechanical sounds to detect problems
- **Understands** failure patterns and recommends intelligent repairs
- **Guides** you through measurements, repairs, and module installations
- **Learns** from every repair to continuously improve diagnostics

## ğŸš€ Getting Started

### Quick Install

```bash
# Clone the repository
git clone https://github.com/Blackmvmba88/iaRealidad.git
cd iaRealidad

# Install dependencies
npm install

# Run on your platform
npm run android   # Android
npm run ios       # iOS  
npm run windows   # Windows
npm run macos     # macOS
```

### First Run

1. Grant camera permissions when prompted
2. Select a mode from the home screen
3. Point your camera at a circuit board
4. Follow the AR overlays and instructions
